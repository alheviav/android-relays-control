# 📱 INICIO RÁPIDO - Compilar APK

## 🚀 Método 1: GitHub Actions (SIN INSTALAR NADA) ⭐ RECOMENDADO

**Lee:** `GUIA_GITHUB.md`

**Resumen:**
1. Sube estos archivos a GitHub
2. Ve a Actions → Run workflow
3. Espera 20 minutos
4. Descarga el APK desde "Artifacts"
5. ¡Instala en tu tablet!

**Tiempo total:** ~25 minutos

---

## 💻 Método 2: Compilación Local con WSL2

**Lee:** `GUIA_COMPILACION.md`

**Resumen:**
1. Instalar WSL2 en Windows
2. Instalar Buildozer
3. Ejecutar `bash compilar.sh`
4. Espera 30-60 minutos (primera vez)
5. Copia el APK

**Tiempo total:** ~80 minutos (primera vez)

## ⚡ Inicio Ultra-Rápido

Si ya tienes cuenta en GitHub:

1. **Sube archivos:**
   - Arrastra todos los archivos a tu repositorio
   
2. **Compila:**
   - Actions → Build Android APK → Run workflow
   
3. **Descarga:**
   - Espera 20 min → Artifacts → Descarga APK
   
4. **Instala:**
   - Transfiere a tablet → Instala

¡Listo! 🎉
